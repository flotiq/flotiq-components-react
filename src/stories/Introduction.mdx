# Welcome to Flotiq Components.

Flotiq components have been designed and implemented to meet the needs of most users. They are all part of a library focused on supporting the development of Flotiq products.

You can build many projects easily and quickly with Flotiq components.

Browse example stories now by navigating them in the sidebar. View their code in the `src/stories` directory to learn how they work.
We recommend building UIs with a [**component-driven**](https://componentdriven.org) process starting with atomic components and ending with pages.
